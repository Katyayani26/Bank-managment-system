Overview
The Bank Management System is a full-stack web application built using Spring Boot for the backend and React for the frontend.
It provides core banking functionalities such as account creation, transaction management, and customer details handling.
The backend uses H2 Database for lightweight storage, making it easy to run locally and deploy on cloud platforms like Render and Vercel.


Features
User-friendly interface for managing bank accounts.

Create, update, and delete customer records.

Deposit and withdraw funds.

View transaction history.

Secure REST APIs built with Spring Boot.

Responsive frontend built with React.

H2 in-memory database for quick setup and testing.
